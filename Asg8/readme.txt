This contains 1 lex code, 1 bison code, 1 input file, 1 output trace, 1 grammar file and 1 makefile.

1. 25.8.l - This is the lex code defining rules for the terminals.

2. 25.8.y - This is the bison code which generates the parser for the given grammar.

3. 25.8.output - This is the file for the output trace with all the comments expalaining the working of the parser on the given input.

4. input.txt - The input source code for the parser.
 
5. grammar.out - This file contains the grammar, terminals, non-terminals and the states of the parser.

6. makefile - The makefile to generate executable file.
